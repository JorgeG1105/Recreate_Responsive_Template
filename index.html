Built with love and attention to detail. This template was created based on another template that was created using Bootstrap. I tried recreating it using no frameworks and anything I didn't know I looked up and applied!

Responsive Design

Responsive design is pretty much a standard nowadays and everything must be mobile friendly. You can rest assured knowing that I built this template and took mobile into consideration. Creating this template also allowed me to get down and dirty with my Flexbox skills which I had been neglecting for a bit.
